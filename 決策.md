
五個要砍的欄位

1. spatial（entry 層級的空間描述）

// 這個 ↓
Look: {
  into
    spatial: "視線穿進內部去看。",   // ← 就是這行
    me
  }
}

它出現在方程式動畫下方的文字。但你已經有：
- VERB

// 這個 ↓
Look:
  into: {
    spatial: "視線穿進內部去看。",   // ← 就是這
    meanings: [...]
  }
}

它出現在方程式動畫下方的文字。但你已經有：
- VERB
- PARTICLE_DIRECTIONS 的 "往裡面衝（動態）"

"視線 句話，等於同一件事說了兩次。

4. pla

meanings: [{
  mean
  plainEnglish: "to check or investigate",   // ← 英文定義                              }]

顯示在 PhraseCard 意思旁邊的藍色小徽章 = to check  經告訴使用者意思了，這個是額外的英文版。

---
5. PhraseCard 推理公式

PhraseCard 最上面的這塊：

Look（用眼睛「看」）+ into（往裡面衝（動態））

但使用者往上看，方程式動畫區已經有一模一樣的資訊（動詞動畫下面寫
Look：  into：往裡面衝）。等於同一行在畫面上出現兩次。
